# Avelot

A UI design trial for Avelot using Svelte.

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/korkordesmond/avelot-ui-svelte.git
cd avelot-ui-svelte 
```  

### 2. Install Dependencies

```bash
npm install
# or
yarn install
# or
pnpm install   
```

### 3. Start Development Server

```bash
npm run dev --- --open  
```

#### Visit http://localhost:5173 to view your app.
